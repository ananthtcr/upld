Support

Bugs & feature requests

If you believe you have discovered a bug in Fine Uploader, please file a bug or feature request in the GitHub issue tracker. Please be sure the issue doesn't already exist. If it does, feel free to comment in the original issue instead. A template will appear once you open a new issue to guide you through the process.

"How do I do...?" or "Why isn't this working?"

For technical support questions, such as "How can I read a modify a file client-side and then send it to Fine Uploader for uploading?", or "Why isn't my onError callback being executed?", please post a question on Stackoverflow under the established fine-uploader tag. If you have never posted a question on Stackoverflow before, read the How do I ask a good question?" guide first. Reading and understanding this guide will save you a lot of trouble and frustration later.

Discussions & developer chat

If you would like to join other developers (including the core team) in discussions about Fine Uploader and other related libraries, join the #fine-uploader Freenode IRC channel. Topics of discussion may include Fine Uploader evolution plans, Modern Uploader development and design, and future integrations (to name a few).
